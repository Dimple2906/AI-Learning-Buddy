# 🎓 AI Learning Buddy - Dimple

An interactive AI tutor that helps students learn any topic using simple explanations, real-life examples, quizzes, and feedback.

## Features

- Explain concepts in simple language
- Provide real-life examples
- Generate MCQ quizzes
- Answer student questions
- Give beginner-friendly learning support

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- Prompt Engineering
